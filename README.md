# Análisis de Sentimientos en Tweets

## 📌 Descripción
Este proyecto implementa un modelo de machine learning para clasificar sentimientos en tweets, determinando si el contenido es positivo o negativo.

---

## 🧠 Modelo
- Tipo: Clasificación de texto
- Problema: Procesamiento de lenguaje natural (NLP)

---

## ⚙️ Tecnologías
- Python
- Pandas
- Scikit-learn
- NLP (procesamiento de texto)

---

## 🔍 Procesamiento
- Limpieza de texto
- Tokenización
- Vectorización (TF-IDF o similar)

---

## ▶️ Cómo ejecutar
1. Abrir notebook en Google Colab
2. Ejecutar celdas
3. Probar con nuevos textos
